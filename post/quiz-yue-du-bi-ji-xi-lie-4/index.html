问题

fn main() {
    let (.., x, y) = (0, 1, ..);
    print!("{}", b"066"[y][x]);
}

解析

NO.1

 let (.., x, y) = (0, 1, ..);

X = 1
Y = ..

b"066"[..][1] = '6'

Rust相关知识点

Rust中..符号有三个含义，官方文档

  • 用作切片索引表示切割完整的数组
    范围：

    示例
  • 结构体表达式
    结构表达式可以以语法..结尾。后跟表示函数更新的表达式。
  • 模式匹配

占位符(_)代表单个数据字段
而通配符…表示特定变体的所有其余字段。
