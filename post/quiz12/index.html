问题

struct S {
    x: i32,
}

const S: S = S { x: 2 };

fn main() {
    let v = &mut S;
    v.x += 1;
    S.x += 1;
    print!("{}{}", v.x, S.x);
}

解析

NO.1

常量表达式在使用时本质上是内联的，这意味着它们在使用时直接复制到相关的上下文中。对相同常量的引用不一定保证引用相同的内存地址。（相等于宏定义）

常量必须显式类型。类型必须具有静态生存期。

const S: S = S { x: 2 };

NO.2

引用类型的任何名称都存在于类型名称空间中，引用值的任何名称都存在于值名称空间中。
这是两组独立的名称，该语言的结构使我们总是能够知道在哪个名称空间中查找名称。

这是在类型空间

struct S {
    x: i32,
}

这是在值空间

const S: S = S { x: 2 };

NO.3

let v = &mut S;
v.x += 1;

相当于

let mut _tmp0 = S { x: 2 };
let v = &mut _tmp0;

NO.4

S.x += 1;

拓展成

S { x: 2 }.x += 1;

NO.5

print!("{}{}", v.x, S.x);

拓展成

 print!("{}{}", v.x, S { x: 2 }.x);
