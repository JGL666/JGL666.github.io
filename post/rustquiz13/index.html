Rust Quiz 解读汇总

struct S;

fn main() {
    let [x, y] = &mut [S, S];
    let eq = x as *mut S == y as *mut S;
    print!("{}", eq as u8);
}

解析

NO.1

S是零大小类型，编译器会进行优化。这里数组的地址和两个元素的地址一样。数组[S; 2]大小也是0。

let [x, y] = &mut [S, S];

Rust相关知识

ZSTS类型：

Rust认为，任何生产或存储ZST的操作都可以被简化为无操作。存储它甚至没有意义——它不占任何空间。可以在编译时期优化代码，较少不必要的内存分配。
