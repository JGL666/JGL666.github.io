问题

macro_rules! m {
    ($($s:stmt)*) => {
        $(
            { stringify!($s); 1 }
        )<<*
    };
}

fn main() {
    print!(
        "{}{}{}",
        m! { return || true },
        m! { (return) || true },
        m! { {return} || true },
    );
}

解析

NO1

表示匹配一个或者多个Rust语句

($($s:stmt)*)

这是Rust匹配宏规则，stmt表示语句

$s:stmt

NO2

Rust在宏中引用$s，表示引用匹配的变量。

stringify!将引用的token转为字符串

{ stringify!($s); 1 }

NO3

return || true 是一个表达式语句 返回一个闭包

 m! { return || true }

如下代码一样

NO4

(return) || true 是一个逻辑或表达式语句

 m! { (return) || true }

NO5

左边是一个块表达式语句，右边是一个闭包表达式语句

这是两个语句

m! { {return} || true }
