fn main() {
    let input = vec![1, 2, 3];

    let parity = input
        .iter()
        .map(|x| {
            print!("{}", x);
            x % 2
        });

    for p in parity {
        print!("{}", p);
    }
}

解析

Map结构体

pub struct Map<I, F> {
    iter: I,
    f: F,
}

Map的方法next

#[inline]
fn next(&mut self) -> Option<B> {
    self.iter.next().map(&mut self.f)
}

可见，迭代调用的是包裹的内部迭代器，这里是数组的next方法，map传递的是Map初始化的函数 &mut self.f 。即：

|x| {
    print!("{}", x);
    x % 2
}
