Rust Quiz 解读汇总

struct D(u8);

impl Drop for D {
    fn drop(&mut self) {
        print!("{}", self.0);
    }
}

struct S {
    d: D,
    x: u8,
}

fn main() {
    let S { x, .. } = S {
        d: D(1),
        x: 2,
    };
    print!("{}", x);

    let S { ref x, .. } = S {
        d: D(3),
        x: 4,
    };
    print!("{}", x);
}

解析

NO.1

生成了临时变量S_temp, 然后采用模式匹配，这里x是值拷贝。
S会立马析构，输出12

    let S { x, .. } = S {
        d: D(1),
        x: 2,
    };
    print!("{}", x);

NO.2

这里x是对临时变量内部字段的引用，临时变量还不能释放，不然会造成悬垂指针。

let S { ref x, .. } = S {
    d: D(3),
    x: 4,
};
print!("{}", x);
